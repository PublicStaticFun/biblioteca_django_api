# Biblioteca API — Django REST Framework
API REST para la gestión de una biblioteca, incluyendo autores, categorías, libros, usuarios, autenticación JWT y sistema de préstamos.

## Introducción
Este proyecto implementa un backend completo para un sistema de biblioteca usando Django REST Framework. Permite administrar:
* Autores
* Categorías
* Libros
* Usuarios
* Préstamos
* Autenticación con JWT
* Filtros, búsquedas y validaciones
* Conexión a base de datos PostgreSQL

## Objetivo del proyecto
El objetivo es ofrecer una API REST funcional que permita:
* Registrar usuarios
* Autenticarlos mediante tokens JWT
* Crear y administrar el catálogo de libros
* Controlar préstamos con reglas de negocio reales
* Aplicar filtros, validaciones, paginación y permisos
* Proveer documentación interactiva para pruebas (Swagger)
